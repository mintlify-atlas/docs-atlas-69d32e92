---
title: 'Sales Report'
api: 'GET /api/Venta/Reporte'
description: 'Generate a detailed sales report for a specific date range'
---

## Query Parameters

<ParamField query="fechaInicio" type="string">
  Start date for the report period.
  
  Format: `dd/MM/yyyy` (e.g., `"01/03/2026"`)
  
  If not provided, returns all sales up to `fechaFin`.
</ParamField>

<ParamField query="fechaFin" type="string">
  End date for the report period.
  
  Format: `dd/MM/yyyy` (e.g., `"05/03/2026"`)
  
  If not provided, returns all sales from `fechaInicio` onwards.
</ParamField>

## Response

<ResponseField name="status" type="boolean">
  Indicates whether the operation was successful
</ResponseField>

<ResponseField name="value" type="array">
  Array of detailed report records. Each record represents a line item from sales in the date range.
  
  <Expandable title="ReporteDTO properties">
    <ResponseField name="NumeroDocumento" type="string">
      Sale document number
    </ResponseField>
    
    <ResponseField name="TipoPago" type="string">
      Payment type for the sale
    </ResponseField>
    
    <ResponseField name="FechaRegistro" type="string">
      Sale registration date and time
    </ResponseField>
    
    <ResponseField name="TotalVenta" type="string">
      Total amount of the sale
    </ResponseField>
    
    <ResponseField name="Producto" type="string">
      Product name or description
    </ResponseField>
    
    <ResponseField name="Cantidad" type="integer">
      Quantity of the product sold
    </ResponseField>
    
    <ResponseField name="Precio" type="string">
      Unit price of the product
    </ResponseField>
    
    <ResponseField name="Total" type="string">
      Line total (quantity × price)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="msg" type="string">
  Error message if status is false
</ResponseField>

## Notes

- The report returns one row per product line item, not per sale
- Each row includes both the line item details and the parent sale information
- This format is ideal for generating detailed sales analysis and product performance reports
- Use for export to Excel, generating charts, or analytical dashboards
