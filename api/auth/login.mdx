## Overview

The login endpoint authenticates users by validating their credentials (email and password) and returns session information including user details and role.

## Request

### Endpoint

```
POST /api/Usuario/IniciarSesion
```

### Headers

<ParamField header="Content-Type" type="string" required>
  application/json
</ParamField>

### Body Parameters

<ParamField body="Correo" type="string" required>
  User's email address
</ParamField>

<ParamField body="Clave" type="string" required>
  User's password
</ParamField>

## Response

### Success Response (200 OK)

Returns a response wrapper containing session information.

<ResponseField name="status" type="boolean" required>
  Indicates if the operation was successful
</ResponseField>

<ResponseField name="value" type="object">
  Session data object
  
  <Expandable title="SesionDTO properties">
    <ResponseField name="IdUsuario" type="integer">
      Unique identifier for the user
    </ResponseField>
    
    <ResponseField name="NombreCompleto" type="string">
      Full name of the authenticated user
    </ResponseField>
    
    <ResponseField name="Correo" type="string">
      Email address of the authenticated user
    </ResponseField>
    
    <ResponseField name="RolDescripcion" type="string">
      Role/permission level description (e.g., "Administrador", "Empleado")
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="msg" type="string">
  Error message (populated when status is false)
</ResponseField>

## Examples

<CodeGroup>

```bash curl
curl -X POST https://api.example.com/api/Usuario/IniciarSesion \
  -H "Content-Type: application/json" \
  -d '{
    "Correo": "admin@example.com",
    "Clave": "password123"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.example.com/api/Usuario/IniciarSesion', {

  headers: {

  },
```

```python Python
import requests

url = 'https://api.example.com/api/Usuario/IniciarSesion'
headers = {'Content-Type': 'application/json'}
payload = {
    'Correo': 'admin@example.com',
    'Clave': 'password123'

response = requests.post(url, json=payload, headers=headers)
print(response.json())
```

```csharp C#
using System.Net.Http;
using System.Text;
using System.Text.Json;

var client = new HttpClient();
var loginData = new {
    Correo = "admin@example.com",
    Clave = "password123"
};

var content = new StringContent(
    JsonSerializer.Serialize(loginData),
    Encoding.UTF8,
    "application/json"
);

var response = await client.PostAsync(
    "https://api.example.com/api/Usuario/IniciarSesion",
    content
);

var result = await response.Content.ReadAsStringAsync();
```


## Error Codes

<ResponseField name="status: false" type="error">
  Authentication failure
  
  **Common error messages:**
  - "Credenciales inválidas" - Invalid email or password
  - "Usuario no encontrado" - User account does not exist
  - "Usuario inactivo" - User account is disabled
</ResponseField>

## Notes

- The endpoint always returns HTTP 200 OK status code. Check the `status` field in the response body to determine success or failure.
- Invalid credentials will return `status: false` with an error message in the `msg` field.
- Store the session information (particularly `IdUsuario` and `RolDescripcion`) for subsequent authenticated requests.
- Passwords are validated securely on the server side.
