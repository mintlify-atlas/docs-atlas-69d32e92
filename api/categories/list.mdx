---
title: List Categories
api: GET /api/Categoria/Lista
---

Retrieves all categories in the system.

## Endpoint

```
GET /api/Categoria/Lista
```

## Request

This endpoint does not require any parameters.

## Response

Returns a `Response<List<CategoriaDTO>>` wrapper containing the list of categories.

<ResponseField name="status" type="boolean" required>
  Indicates whether the request was successful
</ResponseField>

<ResponseField name="value" type="CategoriaDTO[]">
  Array of category objects
  
  <ResponseField name="IdCategoria" type="integer">
    Unique identifier for the category
  </ResponseField>
  
  <ResponseField name="Nombre" type="string">
    Name of the category
  </ResponseField>
</ResponseField>

<ResponseField name="msg" type="string">
  Error message if status is false, null otherwise
</ResponseField>
