---
title: Get Dashboard Summary
api: GET /api/DashBoard/Resumen
---

Retrieves summary statistics for the dashboard including total sales, revenue, products, and sales trends for the last week.

## Endpoint

```
GET /api/DashBoard/Resumen
```

## Response

Returns a dashboard summary containing key metrics and weekly sales data.

<ResponseField name="status" type="boolean" required>
  Indicates if the request was successful
</ResponseField>

<ResponseField name="value" type="object">
  Dashboard summary data
  
  <Expandable title="DashBoardDTO properties">
    <ResponseField name="totalVentas" type="integer">
      Total number of sales
    </ResponseField>
    
    <ResponseField name="totalIngresos" type="string">
      Total revenue/income as formatted string
    </ResponseField>
    
    <ResponseField name="totalProductos" type="integer">
      Total number of products in inventory
    </ResponseField>
    
    <ResponseField name="ventasUltimaSemana" type="array">
      Sales data for the last week
      
      <Expandable title="VentasSemanaDTO properties">
        <ResponseField name="fecha" type="string">
          Date of the sales record
        </ResponseField>
        
        <ResponseField name="total" type="integer">
          Total number of sales for that day
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="msg" type="string">
  Error message if status is false
</ResponseField>

## Example Request

<CodeGroup>
```bash cURL
curl -X GET "https://api.example.com/api/DashBoard/Resumen" \
  -H "Content-Type: application/json"
```

```javascript JavaScript

  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }

const data = await response.json();
```

```csharp C#
using var client = new HttpClient();
var response = await client.GetAsync("https://api.example.com/api/DashBoard/Resumen");
```
